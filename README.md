# Password Strength Checker

## Domain
Cybersecurity & Ethical Hacking

## Description
This is a beginner-friendly Python GUI tool to check the strength of passwords.
It helps users understand how secure their passwords are and suggests improvements.
This project is created for educational purposes as part of RISE 3.0 Internship.

## Features
- Checks minimum length (8+ characters)
- Checks for uppercase, lowercase letters, numbers, and special characters
- Avoids commonly used passwords
- Displays password strength: Weak / Medium / Strong
- Provides suggestions to improve weak passwords
- Simple GUI using Tkinter for easy interaction

## Tools & Technologies
- Python 3.12
- Tkinter (GUI Library)
- Regex (Pattern matching)
- Command Line (CMD) for running code

## Installation Instructions
1. Ensure Python 3.12 is installed: [Python 3.12 Official Download](https://www.python.org/downloads/release/python-3120/)
2. Clone or download this repository
3. Navigate to the project folder in CMD/Terminal
4. Run the application:
5. 5. GUI window will appear, enter your password to check its strength
